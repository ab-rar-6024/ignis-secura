REQUIRED ASSETS — place these files here:
==========================================

1. logo.png             → Your app logo (512x512 px, transparent bg recommended)
2. alarm.mp3            → Alarm sound file (loops on leak detection)
3. icon.png             → App icon for stores (1024x1024 px)
4. adaptive-icon.png    → Android adaptive icon foreground (1024x1024 px)
5. splash.png           → Splash screen (1284x2778 px recommended)
6. notification-icon.png → Android notification icon (96x96 px, WHITE on transparent)
7. favicon.png          → Web favicon (48x48 px)

Where to get free assets:
- Logo: design at canva.com or figma.com
- Alarm sound: freesound.org (search "alarm" → download MP3)
- Icons: use expo's default or generate at appicon.co
